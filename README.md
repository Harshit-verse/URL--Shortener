🔗 URL Shortener
🚀 About

A simple yet powerful URL shortener built with HTML, CSS, JavaScript, and Node.js.
Transforms long, messy URLs into clean, shareable links with fast redirection and a minimal UI.

Think: less clutter, more efficiency 💡

✨ Features
🔗 Generate short URLs in seconds
⚡ Lightning-fast redirection
🎯 Clean & minimal frontend (no distractions)
🧠 Efficient backend handling with Node.js
📦 Lightweight and easy to run locally
🛠️ Tech Stack
🎨 Frontend: HTML, CSS, JavaScript
⚙️ Backend: Node.js
🚀 Runtime: Node.js
⚙️ How It Works
User Input → Server → Short ID Generation → Store Mapping → Redirect
🧑‍💻 User enters a long URL
⚙️ Server generates a unique short ID
🔗 Short URL is created & mapped to original link
🚀 Visiting short URL redirects instantly
📂 Project Structure

🧪 Installation & Setup
# 📥 Clone repo
git clone https://github.com/your-username/url-shortener.git

# 📂 Move into project
cd url-shortener

# 📦 Install dependencies
npm install

# 🚀 Start server
npm start
